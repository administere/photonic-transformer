## 1. 系统架构回顾

### 1.1 计算模型

光子注意力加速器用 MZI 网格实现 Transformer 中的缩放点积注意力：

```
Attention(Q, K, V) = softmax(QK^T / √d_k) V
```

- **QK^T 点积**: 通过电光调制将数字点积编码为 MZI 相位差 Δφ
- **softmax 近似**: MZI 透射率经行归一化替代指数函数
- **V 加权求和**: 后续通过探测器阵列完成

### 1.2 器件结构

```
输入 ──→ [MZI PP] ──→ [WDM demux] ──→ 探测器
       └─ [MZI PN] ──┘
```

- 单个 MZI: 两个 3dB 定向耦合器 (DC) + 热光移相器
- DC 耦合长度: ~10 μm (Meep FDTD 验证)
- WDM 通道: C 波段 (1.53–1.57 μm), 微环解复用

### 1.3 关键设计参数

|------|-----|
| 硅折射率 | 3.477 @ 1.55 μm |
| SiO₂ 折射率 | 1.444 |

| DC 间隙 | 200 nm |
| 相位调制效率 | β = 1.0 rad/(a.u.) |
| 工作偏置点 | π/2 (正交) |

### 3.3 成品率评估

**结论: 在最恶劣工艺角下，500 次蒙特卡洛零失效。成品率 100%（95% CI: [99.26%, 100.00%]）。注意力排序对制造偏差高度鲁棒。**

统计解释：100% 是点估计。使用 Clopper-Pearson 精确二项置信区间，在 N=500 且 0 次失效的情况下，真实失效率在 95% 置信度下不超过 0.74%。N=500 提供了合理的统计可信度，但更大的 N 值（如 N=5000）可进一步收紧置信区间。

根本原因分析:
1. MZI 传输函数在 [0, π] 区间保持单调 → 工艺偏差不会改变符号方向
2. softmax 归一化对单调变换不变 → 排序保持
3. 四种偏差源独立且量级有限 → 总效应仍在线性区域

### 5.6 结论

**热串扰是可量化和可解决的。** 在 20 μm 未缓解间距下，热串扰使 Spearman ρ 降低约 0.047（至 0.9495），属于显著但可管控的退化。4 种缓解策略中有 17/19 个方案实现成品率 100%。

**推荐方案**（按实用性排序）:
1. **浅保护环沟槽（2μm）在 30μm 间距** — 工艺最简单，面积成本适中

4. **主动补偿作为补充** — 增加控制复杂度，最好与物理隔离组合使用

## 7. 下游任务影响评估

### 方法

数字训练 → 光子推理 (最接近实际部署场景):
1. 用标准 softmax 注意力训练 2 层 Transformer (d=64, 4 heads)
2. 推理时将注意力替换为 MZI 光子网格
3. 在合成二分类任务上测量准确率
4. 比较含/不含工艺噪声的光子推理

### 结果

| 配置 | 准确率 | Δ vs 数字推理 |
|------|:------:|:------------:|
| 数字训练 + 数字推理 | 67.30% | — (baseline) |
| 数字训练 + 光子推理 (clean MZI) | 64.00% | +3.30% |
| 数字训练 + 光子推理 (含工艺噪声) | 63.96% | +3.34% |
| **工艺噪声 Δ (clean − noisy)** | | **+0.04%** |

### 结论

1. **工艺噪声影响可忽略**: 含/不含工艺噪声的光子推理准确率差异仅 0.04%，远小于 0.5% 阈值。✅
2. **架构差距可控**: exp-softmax → MZI-sin² 的转换导致 ~3.3% 绝对准确率下降，但这是非线性激活函数差异引起的，非器件非理想性。该差距可通过以下方式消除：
   - 光子感知训练 (photonic-aware training)
   - 知识蒸馏 (teacher: digital, student: photonic)
   - 部署前微调 (fine-tuning on photonic model)
3. **工艺偏差不造成额外退化**: 10 次含噪声推理标准差仅 0.0021，证实蒙特卡洛分析的结论。
